What is python

Python is a high-level, interpreted, general-purpose programming language known for its clear and readable syntax. It was created by Guido van Rossum and first released in 1991.

Why learn python

  • It is easy to learn
  • General purpose programming language
  • It is a dynamic type programming language

Applications of python

  2. Data science
  3. Automation
  4. Software testing
